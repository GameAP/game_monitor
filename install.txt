Gmon v1.0 for GameAP
Модуль мониторинга игровых серверов. Поддерживает огромное количество игр (более тысячи).
Модуль достаточно прост, как в использовании, так и в установке. Сделан для демонстрации внутренних возможностей GameAP, служит скорее платформой для создания своего, более расширенного модуля.

Требуемая версия GameAP: 0.9 и выше
Скачать панель GameAP можно на официальном сайте www.gameap.ru


----------- Установка -------------

1. Скопируйте директорию application из архива в корневую директорию с панелью на сервере.

2. Обновите список модулей. Для этого зайдите в панель и перейдите 
Модули -> Управление модулями -> Обновить список модулей

3. Откройте файл конфигурации application/modules/gmon/config/gmon_config.php и настройте по своему усмотрению.
